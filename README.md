# Minireto de acceso a datos
Es un programa para gestionar *videojuegos*, se pueden dar de alta, eliminar y guardarlos en un archivo txt.

## Datos sobre cada videojuego:
+ Titulo
+ Consola
+ Estudio
+ Tiempo de speedrun
+ Speedruner
+ Año de estreno
